This is a symon says game. where we can play this game in free time. This is so interested. game. try it. This is developed by me with help of HTML CSS JavaScript and Bootstrap these technologies.

If you’re looking for a fun game that’s perfect for kids and adults, Simon Says is the perfect thing to play. Simon Says is perfect for improving listening skills for children, and it’s super easy to learn for the whole family. Keep reading, and we’ll walk you through how to play as well as some exciting variations to try out! Choose a player to be Simon. When Simon gives a command starting with “Simon says…” the other players have to follow the instructions. If a player doesn’t follow the instructions correctly, they’re out of the game. The last player standing is the winner!
